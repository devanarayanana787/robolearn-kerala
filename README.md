## Deploy to Vercel

### Option 1: Drag & Drop
1. Go to [vercel.com](https://vercel.com)
2. Sign up / Login
3. Click "Add New Project"
4. Drag this folder to upload
5. Click "Deploy"

### Option 2: Git Integration
```bash
# Initialize git
git init
git add .
git commit -m "Initial commit"

# Push to GitHub
git remote add origin https://github.com/YOUR_USERNAME/robolearn-kerala.git
git push -u origin main
```
Then import the GitHub repo in Vercel.

### Option 3: Vercel CLI
```bash
npm i -g vercel
vercel login
vercel --prod
```
